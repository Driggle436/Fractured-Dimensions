# Fractured-Dimensions
A 2D Platformer made in Godot about making use of multiple hidden dimensions to solve puzzles and defeat bosses

# Group Project
This project was developed as part of a team collaboration for Inter IIT.

## My Contribution
- Implemented dimension switching system (WorldSwitcher)
- Designed Level 2 puzzles (bridge, rubble, invisible path)
- Developed enemy mechanics (movement and statue behavior)
- Implemented background switching for Normal and Hell worlds
